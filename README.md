# E-Commerce Conversion Insights

## 📌 Project Overview
This project focuses on analyzing customer engagement, conversion behavior, and purchasing trends in an e-commerce platform using Power BI and Python. The dashboard provides insights into customer activities such as product views, cart additions, purchases, revenue generation, category performance, and brand analysis.

---

## 🎯 Objectives
- Analyze customer conversion behavior
- Monitor customer purchase trends
- Identify top-performing brands
- Visualize category-wise customer purchases
- Track customer engagement metrics
- Build interactive dashboards using Power BI

---

## 🛠 Tools & Technologies Used
- Power BI

- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Key Insights
- Product views are significantly higher than completed purchases.
- Electronics and smartphone categories dominate customer engagement.
- Certain brands generate higher revenue compared to others.
- Customer activity remains active across the analyzed period.
- The conversion funnel highlights customer drop-offs between cart additions and purchases.
